The Resource Management Tool is an application built to solve our companies internal business needs. Leveraging the OpenAir API, it takes tools and processes that are already in place, makes them more efficient by creating a better user experience, enhances them by adding new features, which then creates more visibility into ways our company can improve. Key features built so far are the creation and management of new projects, resourcing roles to those projects efficiently by using user profile's and a simple algorithm, and using data visualization to see how resources are being utilized. Key contributions include leveraging VueJS and ChartJS to visualize how our resources are being utilized, caching data and storing session information on the back end using redis, and creating a continuous deployment pipeline using Jenkins and Docker.
